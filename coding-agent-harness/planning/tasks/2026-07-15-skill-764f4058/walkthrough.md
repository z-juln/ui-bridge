# 收口记录：实现通用 Skill 安装教学入口

## 摘要

`ui-bridge-control` 通用 Skill 已随 UI Bridge.app 打包。连接页提供 Agent 中立的教学按钮，用户可以
预览并复制简短安装提示词，再交给当前 Agent 安装；运行规则只保留在 `SKILL.md`。
Skill 名称不再绑定 macOS；当前发行版的能力范围仍明确为 macOS，未来 Windows 实现沿用同一名称。
