# 实现完整设置与实时调试界面

## Task ID

`2026-07-14-item-65db687f`

## 一句话结果

UI Bridge 提供可直接打开的完整设置窗口，并实时展示所有正在被操控应用的画面、来源、动作与安全状态。

## 完成后能得到什么

用户可以从菜单栏或程序坞打开设置窗口，查看服务、权限、连接、应用访问和诊断状态；在“实时操控”中看到所有当前活动应用的实时缩略图和选中应用的大画面，以及操控来源、动作、控件位置和事件记录。删除、购买、权限变更等危险动作必须在 App 内再次确认，拒绝或超时均不得执行。

## 交付物

- 可见产物：原生 macOS 设置与调试窗口、实时画面、安全确认弹窗。
- 修改位置：`Sources/ui-bridge/`、必要的核心活动记录与协议入口、`docs/`。
- 验证证据：构建、自检、安装版真实界面截图和交互记录。
